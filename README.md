# Kickresume Job Board


## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- TanStack Query v5
- Zod (API response validation)
- react-error-boundary

## Setup

```bash
npm install
npm run dev
```

## Scripts

- `npm run dev` — Start development server
- `npm run build` — Production build
- `npm run lint` — Run ESLint
- `npm run preview` — Preview production build
